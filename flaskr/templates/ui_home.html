Collaberative Filtering

When the reccomendations are generated, we measure the similiarity between users. We formulate that measurement using the Pearson Correlation Similiarity Measure which provides the nessesary users for an estimate. In order to improve the reccomendations, users and items are clusterd using the KMeans Clustering Algorithm. Using this technique, the means squared error can minnimized, even with minimal data.
